Leetcode-每日一题-lc1025除数博弈

七月 24, 2020 发布在 算法与数据结构, leetcode每日一题

题目描述(leetcode-1025)

难度：简单

爱丽丝和鲍勃一起玩游戏，他们轮流行动。爱丽丝先手开局。

最初，黑板上有一个数字 N 。在每个玩家的回合，玩家需要执行以下操作：

选出任一 x，满足 0 < x < N 且 N % x == 0 。
用 N - x 替换黑板上的数字 N 。

如果玩家无法执行这些操作，就会输掉游戏。

只有在爱丽丝在游戏中取得胜利时才返回 True，否则返回 false。假设两个玩家都以最佳状态参与游戏。

示例 1：

输入：2
输出：true
解释：爱丽丝选择 1，鲍勃无法进行操作。

示例 2：

输入：3
输出：false
解释：爱丽丝选择 1，鲍勃也选择 1，然后爱丽丝无法进行操作。

提示：

1 <= N <= 1000

思路分析:

 对状态进行分析，可以知道N的状态是由（N-x）得到的，利用动态规划思想，$dp[i]$表示当前数字是$i$时，Alice能否获胜，获胜为True，失败为False。在当前状态获胜的条件为，可以找到一个子状态其是失败的，也就是可以通过操作得到这个子状态，从而让Bob失败，即Alice获胜。可以写出递推方程：

$$dp[i]=(dp[i-j] == false) && (i%j==0)$$

最终算法的时间复杂度为O($n^2$)，空间复杂度为O(n)

P.S. 看过解答后，这题其实是一个数学题，可以根据数学归纳法，进行求解
